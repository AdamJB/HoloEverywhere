What is it?
Backport of Holo Theme from Android 4.1 to 2.1.
Your application can use a lot of new features on older platforms!

Modal choice mode for ListView? Easy!
Dropdown popup for Spinner selector? Easy!
Preference headers? Easy!

Just write code - we're stylize it.
Addon's system
Remember those awful SherlockFragmentActivity, RoboguiceActivity or mega-hybrid: SlidingRoboguiceSherlockPreferenceActivity?
Put an end to this!

Now there's only Activity. It can attach to itself additional modules such as ActionBarSherlock or Roboguice.
And no FragmentActivity - fragments support are everywhere - i.e. in ListActivity you can use DialogFragment, that was previously to be difficult with the support library.
